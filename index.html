Introduction

For this project, I wanted to look at weather data in Utah County (as seen above) and predict when it would rain based off of the collected data. I gathered data from the National Center for Environmental Information (https://www.ncdc.noaa.gov/cdo-web/). This data shows the amount of precipitation recorded each day from 1970-2022.

Creating a Predictive Model Using the Prohpet Package

I fed my data into a program called Prophet (made by Facebook) to create predictions of future amounts of precipitation based off historical data.

This doesn’t look like much, but this graph is showing how much data the Prophet program is analyzing as well as a short prediction of future amounts of precipitation in Utah County. Since day-to-day, the amount of precipitation is between 0-1 inches, the data is hard to visualize in a large graph like this.

Another tool from the Prophet package is to show model-created trends in precipitation over the year and over time.

Conclusion

I’ve managed to wrangle real-life data into a workable data set that shows the changes of precipitation in Utah County for over 50 years. I’ve used the Prophet package to predict the rainfall amounts in the following year as well as displayed learned trends of rainfall yearly in Utah County.
